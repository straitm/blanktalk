\section{Intro}

\ul

\item LIGO/Virgo (LVC) has reported 7 gravitational wave events

\item 6 black hole mergers (starting to be routine!)

\item 1 neutron star merger

\item Other possible signals: supernovae, exotics

\item We should report on whether we saw anything coincident
with any of these

\lu

\ul

to search for anything unusual in a $\{-500, +500\}$ second window around
the event

\frame{\frametitle{FD supernova-like event search}

Each hit:

\ul

  \item Between 95 and 600 ADC

  \item More than two planes from either end

  \item $y$ view: more than 50 cells from the top

  \item $x$ view: more than 10 cells from each side

  \item Separated in time \& space from all slices. For each, rejected if it is:
  
  \ul 
     \item Between 4\,$\mu$s prior to the beginning and 13\,$\mu$s after the end, and

     \item Within 14 planes of the maximum plane extent, and

     \item Within 28 cells of the maximum cell extent in the view of the hit
  \lu

\lu

\cols

\col

Pair of hits must:

\ul

  \item Be within 200\,ns of each other, using fine timing (\texttt{CellHit::TNS()})

  \item Be exactly one plane apart

  \item Not be part of any other selected pair

\lu

\section{Conclusions}

\frame{\frametitle{Commentary}

\ul

\item This is largely an exercise to prepare for future events

\item Tiny livetimes mean no chance of seeing a low-energy burst, especially 
since Super-K, Borexino and KamLAND have already reported null results

  \ul

  \item In the future, we can trigger on LVC events and get 100\% livetime

  \lu

\item For triggered events, not as hopeless, but only
within phase space not excluded by Super-K and IceCube

\lu

}
